Article Tag

Per w3schools.com: The article tag specifies independent, self-contained content. It stands on its own. It does not have to relate to other parts of the web site.

Details Tag

This is used often to create an interactive widget that a user can open and close. By default, the widget is closed. The summary tag is used in conjuction with the details to spoecify a visible heading for the details.

time tag and datetime attributes:

The time tag defines a specific time (or datetime).

The datetime attribute of this element is used to translate the time into a machine-readable format so that browsers can offer to add date reminders through teh user's calendar, and search enines can produce smarter search results.

Example: Open from 09:00 to 18:00 every weekday.

Example: datetime: I have an apointment on Sunday.
